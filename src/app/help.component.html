Class Tracker

Useful tips for getting started

This class tracker is designed to assist and accelerate your class preparation. It should not be used as a replacement for your standard preparation but instead to enhance it. Class tracker should be able to speed up your preparation by giving you fast access to your classes, kids, the lessons they will have and the objectives, key words, sentences, and extra activity ideas for those lessons. A schedule is also provided.

With the main navigation bar you can reach the pages for classes, lessons, your schedule, this page, and the login/logout page

Adding Classes

The first step to making full use of the class tracker is to add your classes

First select the "Classes" tab, then choose the "Add Class" button

Here you can add all the details of your set class.

Please fill in the whole form as it will help you to add classes to your class tracker schedule later.

Done! you have added your first class.

You can now continue to add further classes to your list.

When you have added 1 or more classes you can proceed to add kids and lessons to your class.

Adding Details To Your Classes And Editing Classes

Once you have added a class you can add lessons and kids to the class. Adding all the kids is useful for checking names, and for seeing the maximum number of kids who can show up to the class for craft/worksheet preparation purposes. Having the most recently taught classes as well as upcoming classes can help to check what the kids should know, which is useful for warm up games, and what lessons they have coming up.

Click the "Add Kids" button to start adding kids to the class.

The only required detail is the name, but you can provide as much detail as you like (you can edit this later by clicking on the kid in the list).

You can continue to add as many kids to the class as needed.

If a kid leaves the class you can remove them from the list with the "Remove" button next to their name

At the bottom of the class page is the "Add next lesson details" button for adding lessons to the class.

Using this button you can easily add lessons by subject and lesson number, and also select the time and day that the lesson will be taught (you can select from any of the session times you already provided when creating the class).

If anything about the class changes (they graduate to a new stage, or the times of their classes change) you can simply select the edit class details button to update their information. Finally if a class leaves, or is transferred to another teacher as a set class, you can remove the class from your list with the "Remove Class" button.

Adding Missing Lessons And Editing Existing Lessons

Many lessons have been added already to the class tracker, especially those in the lower stages, however the list is not complete. Any one who is logged into the app can add new lessons and edit the existing lessons. Before adding new lessons please check some older lesson details to see how they are layed out. You can also edit lessons, and add any new activity ideas or comments to the activities section in the details, please add all the ideas you have used successfully in your classes!!

You can add any lesson that does not exist yet.

At first it is best to copy the basic details from the teaching guide, then you can add more ideas or comments as you go.

Try to add details in a way that makes them easy to read.

If you add any extra activities/ideas please make sure to note down the slide(s) that they apply to, and add any necessary materials to the supplies list.

If you want to you can also put your name next to the activities so everyone will know who to thank for them.

Editing a lesson follows the same process as adding a lesson, however not all details are editable, so if the lesson has been added with a lesson title and lesson number that don't match, or added to the wrong subject the lesson will need to be deleted and readded

The Schedule

This schedule will show all the lessons you have added to your classes, for this week, last week, and (if you receive your schedule in time) next week.

You can click on these lessons to go directly to the class the lesson is with and check details of kids and keywords etc. Lessons that you have added to your set classes will automatically be added to your schedule.

When you have added all the lessons to your schedule for this week you can download a .xlsx excel spreadsheet to edit as you like that includes many of the class details.

If you have a demo or a class that is not in your set class list, you can also add these with the "Add Irregular Lesson" button.

You will need to add details of the class, including the class code, lesson, and the lesson time and length.

If you click an irregular lesson you will be taken to the lesson page where details of all added lessons are held so that you can find and check the details.
